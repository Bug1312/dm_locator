# DM Locator Addon
A [Dalek Mod](https://modrinth.com/mod/dalek-mod) addon that adds in a the ability to locate biomes and structures with a new attachment on the flight panel!

Craft yourself a **Locatable Data Writer** to write data modules in your inventory for structures or biomes. Shift-Right-Click to swap writing mode.

Use the **Locator Attachment** by placing it on the Flight Panel to gain the ability to locate waypoints, structures, and biomes in flight mode for the cost of artron *(waypoints do not cost fuel)*.
